# 属性养成器 小程序（最小可运行原型）

本仓库包含一个微信小程序的最小原型，实现了：
- 聊天页（模拟 AI 回复，关键字触发属性经验）
- 属性面板（显示 8 项能力及经验进度）
- 任务页（新增任务、完成任务并发放经验奖励）
- utils/storage.js 本地存储与经验发放逻辑
- utils/ai.js 模拟 AI 接口（示范 cloud 函数对接点）

如何运行
1. 打开「微信开发者工具」，选择「小程序项目」并将项目目录设置为本仓库根目录 `d:/miniprogram`。
2. 点击编译预览。首次运行会写入默认属性到本地存储。

下一步建议
- 将 `utils/ai.js` 替换为通过云函数调用真实大模型接口（参考注释）。
- 增加能力升级动画、经验获取记录和排行榜功能（云端）。
- 用云开发数据库替换本地 storage，以支持多端同步与排行榜。

说明
- 升级规则示例：每 100 经验自动 +1 级（可在 `utils/storage.js` 中调整）。
